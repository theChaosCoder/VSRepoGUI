# VSRepoGUI
A GUI for VSRepo

You'll need [listpedeps.exe](https://github.com/brechtsanders/pedeps/releases) for missing plugin dependencies. 

Paste it in the same folder as vsrepogui.exe or install with `choco install pedeps`
